About WikiProvenance

What is WikiProvenance?

WikiProvenance is a web-based tool for exploring and analyzing source statistics for Wikidata items. It provides comprehensive insights into references, external identifiers, and cross-wiki links, helping researchers, editors, and enthusiasts understand the provenance and documentation quality of Wikidata content.

Built with modern web technologies and powered by the Wikidata and MediaWiki APIs, WikiProvenance offers an intuitive interface for discovering how well Wikidata items are sourced and documented across the Wikimedia ecosystem.

Core Features

WikiProvenance provides comprehensive statistics and analysis tools:

Reference Statistics
View the percentage of statements with references for any Wikidata item
External Identifiers
Count and explore connections to external databases and authority files
Cross-Wiki Links
Track articles across Wikipedia, Wikinews, Wikibooks, Wikisource, and more
Item Comparison
Side-by-side comparison of source statistics for multiple items
Smart Search
Quick search functionality to find and analyze any Wikidata item
Bookmarkable Links
Share direct links to specific items and comparisons

Technologies

WikiProvenance is built as a lightweight, client-side application that can run in any modern browser. It requires no server-side infrastructure and can be easily deployed or integrated into existing websites. The application uses:

HTML5 CSS3 JavaScript JSON SPARQL Wikidata API MediaWiki API

The application is designed to be accessible, performant, and easy to deploy on any platform supporting static HTML files, making it ideal for both desktop and mobile environments.
